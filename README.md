## Course Purpose

The course covered the fundamentals of Java development, including:

- How Java code is compiled and how the JVM works

- Working with collections (arrays, lists, sets, stacks, maps)
- Organizing code with packages and namespaces
- Handling exceptions and using annotations
- Writing unit tests with JUnit
